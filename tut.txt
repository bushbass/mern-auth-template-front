backend
    /users/register     POST    auth-no     returns {_id:id,email,string,password:hash,displayname:string}
    /users/login        POST    auth-no     returns {token:string,user:{id:id,displayname:string}}
    /users/delete       DELETE  auth-yes    returns user object same as above
    /users/tokenIsValid POST    auth-yes    returns boolean
    /users              GET     auth-yes    returns current logged in user

    /todos              POST    auth-yes    returns todo -> {_id:string,title:string,userId:string} 
    /todos/all          GET     auth-yes    returns array of todos 
    /todos:id           DELETE  auth-yes    returns deleted todo

frontend
    App.js
        useEffect - checks if there is a token in local storage
                    checks if it is a valid token 
                    gets current user object {displayname:string, id:string}
                    adds current user object and token to state 'userData'
        <UserContext.Provider value={{ userData, setUserData }}>
                    provides access to userData to all child component
        userData = {token:string, user:{id:string,displayname:string}}
